# Basic Programming Assignments (C Language)

This repository contains a collection of programming assignments and lab exercises for the **Basic Programming** course (Semester 1). The projects are organized into 6 modules, covering fundamental concepts of the C programming language.

## 📂 Project Structure

The project is structured into 6 main modules:

*   **Modul 1: Fundamentals** - Basic I/O, arithmetic operations, and data types.
*   **Modul 2: Control Flow** - Conditional statements (`if-else`, logic).
*   **Modul 3: Intermediate Logic** - Further practice with control structures.
*   **Modul 4: Loops & Strings** - Iteration (`while`, `for`) and basic string manipulation.
*   **Modul 5: Arrays & Sorting** - Multi-dimensional arrays and alphabetical sorting algorithms.
*   **Modul 6: File Handling** - Persistent data storage (CRUD) for an "Electronic Shop Management" system.

## 🚀 Getting Started

### Prerequisites

To compile and run these programs, you need a C compiler installed on your system, such as:
*   [GCC (MinGW for Windows)](https://www.mingw-w64.org/)
*   [Clang](https://clang.llvm.org/)
*   An IDE like Dev-C++, Code::Blocks, or VS Code with C/C++ extension.

### Compilation

You can compile any `.c` file using GCC in your terminal:

```bash
gcc "Modul 6/Kegiatan1.c" -o "Modul 6/Kegiatan1"
```

### Running the Program

After compilation, run the generated executable:

```bash
./"Modul 6/Kegiatan1"
```

## 🛠️ Key Features (Modul 6)

The final module features a robust **Electronic Shop Management System** that supports:
- **Adding new products** to a database (`.txt` file).
- **Listing all products** stored in the file.
- **Updating product details** by ID.
- **Deleting products** from the system.
